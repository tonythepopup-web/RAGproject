Chapter 1. 안전관리인증기준(HACCP) 개요

1

안전관리인증기준(HACCP) 이란

안전관리인증기준(HACCP, 해썹)이란 식품의 원재료부터 제조, 가공, 보존, 유통, 조리단계를 거쳐 최종소비자가 섭취하기 전까지의 각 단계에서 발생할 우려가 있는 위해요소를 규명하고, 이를 중점적으로 관리하기 위한 중요관리점을 결정하여 자율적이며 체계적이고 효율적인 관리로 식품의 안전성을 확보하기 위한 과학적인 위생관리체계라고 할 수 있습니다. ☞ 업체에서 사용하는 모든 원·부재료와 각 제조공정별로 발생 가능한 위해요소를 분석하고 관리 계획에 따라 사전 예방하는 것

해썹(HACCP)은 전 세계적으로 가장 효과적이고 효율적인 식품 안전 관리 체계로 인정받고 있으며, 미국, 일본, 유럽연합, 국제기구(Codex, WHO, FAO) 등에서도 모든 식품에 해썹를 적용할 것을 적극 권장하고 있습니다.

HA(위해요소 분석): 안전에 영향을 줄 수 있는 위해요소와 이를 유발할 수 있는 조건이 존재하는지 여부를 판별하기 위하여 필요한 정보를 수집하고 평가하는 일련의 과정

CCP(중요관리점): 위해요소를 예방 및 제어하거나 허용 수준 이하로 감소시켜 당해 식품 및 축산물의 안전성을 확보할 수 있는 중요한 단계, 과정 또는 공정

<생물학적 위해요소> 세균, 바이러스, 기생충 등(살모넬라, 장출혈성대장균 등 주요 식중독균)

<물리적 위해요소> 쇠, 머리카락, 나무토막, 철, 칼날 등

<화학적 위해요소> 농약, 중금속, 항생제, 불완전 연소물 등

2

선행요건관리란 무엇인가요?

“선행요건(Pre-requisite Program)”이란 「식품위생법」, 「건강기능식품에 관한 법률」, 「축산물 위생관리법」에 따라 안전관리인증기준(HACCP)을 적용하기 위한 위생관리프로그램을 말합니다.

안전관리인증기준(HACCP)을 적용하고자 하는 업소의 영업자는 축산물위생관리법 등 관련 법적 요구사항을 준수하면서 위생적으로 제품을 생산하기 위한 기본시스템을 갖추기 위해 작업 기준 및 위생관리기준을 포함하는 선행요건관리를 먼저 개발하여 시행해야 합니다.

- 위생관리

- 영업장관리

- 회수 프로그램 관리, 검사관리

- 보관 및 운송 관리

- 용수관리

- 냉장 및 냉동 시설 설비 관리

- 제조, 가공 선별 포장 시설 설비 관리

3

HACCP관리란 무엇인가요?

HACCP 관리는 7원칙 12절차에 따라 업소에 맞는 기준을 수립하여 운영합니다. 7원칙 12절차는 준비 5단계와 HACCP 7원칙으로 이루어져 있습니다.

<준비5단계>

HACCP팀 구성 -> 제품 설명서 작성 -> 용도 확인 -> 공정흐름도 작성 -> 공정 흐름도 현장 확인

<HACCP 7원칙>

위해요소 분석 -> 중요관리점(CCP) 결정 -> 한계기준 설정 -> 모니터링 체계 확립 -> 개선조치 방법 수립 -> 검증 절차 및 방법 수립 -> 문서화 및 기록유지

Chapter 2. HACCP 인증신청 절차

1

HACCP 인증 절차 간소화 모식도

해썹(HACCP) 인증 신청 및 현장실사는 아래와 같은 순서로 이루어지고 있습니다. 영업 인·허가 사항 및 품목제조보고 완료 여부 확인이 최우선이며, 부득이한 사유를 제외하고는 구비서류 준비 전 HACCP 교육수료가 필요합니다.

1) 인증 신청·접수 (신청 업체)

- 식품 또는 축산물 제조·가공업 영업 등록·허가를 해당 지자체(시·도 또는 시·군·구)에 신청

- 영업 인·허가 신청 시 지자체에서 'HACCP 인증 신청 확인증' 제출을 요구할 경우 인증원 관할 지원에 연락하여 확인증 발급 신청(이메일 신청, 33p 붙임1 참고)

- 식품안전나라 홈페이지(www.foodsafetykorea.go.kr)에서 영업 인·허가 완료 여부 확인 가능(34p 참고)

- 영업 허가(등록)증을 교부 받았으나 식품안전나라 홈페이지에서 업체검색이 안될 경우, 홈페이지 관리부서에 연락하여 적절한 조치 요청(☎1899-5590)

2) HACCP 교육 수료 (신청 업체)

- 식품/축산물 교육 수료 요건이 다르며 신청하고자 하는 업종에 해당하는 교육을 이수하여야 함(12p 상세 설명)

예시) 식품제조가공업-식품 HACCP 교육(식품위생법) / 축산물가공업-축산물 HACCP 교육(축산물위생관리법)

- HACCP 교육은 일정, 신청방법, 교육내용 등이 교육기관마다 다르며 접수 조기 마감에 유의(13p 상세 설명)

- 인증원 교육은 온라인/오프라인 신청 가능

- 연간 교육일정은 각 교육기관 홈페이지에서 확인 가능

- 축산물 영업자(4시간), 종업원(24시간) / 식품 영업자(2시간), HACCP팀장(16시간)

3) 구비서류 준비 (신청 업체)

- HACCP 인증 신청을 위한 구비서류를 준비(15p 참고)

- 구비서류 중 한 개라도 누락이 되면 정상접수 되지 않으며 추가서류 준비·제출 과정 중 수일이 소요되어 최종 접수가 늦어질 수 있음

- 소규모 신청 시 증빙 서류 필수 제출(직전연도 매출액, 연평균 종업원 수 증빙 필요)

4) HACCP 인증 신청 (신청 업체)

- 온라인, 이메일 등의 방법으로 심사 신청 가능(23p 상세 설명)

- 구비서류 누락 없이 첨부하여 신청

- 인증 신청 2주 후에도 수수료 안내 문자 또는 추가 서류제출 요청 등 별도 안내가 없을 경우 해당 지원에 연락하여 정상 접수 여부 확인

5) 신청 서류 접수 (인증원)

- 업체에서 제출한 신청 서류를 인증원에서 접수

- 서류 미비 시 추가 서류 제출을 요청(인증원→업체)

6) 심사 수수료 안내 (인증원→업체(문자))

- 서류 접수가 완료된 업체에 한해 심사 수수료 문자 안내

- 신청 서류에 적혀있는 휴대전화 번호로 문자 알림이 가며 서류접수 완료와 동시에 문자가 자동으로 발송되는 시스템

7) 심사 수수료 입금 (업체→인증원(계좌이체))

- 수수료 안내 문자에 적혀있는 계좌번호로 입금

- 수수료 입금은 계좌이체만 가능(카드 불가), 입금 확인 후 계산서 발급 가능(계산서 선발급 불가)

- 입금자는 반드시 업체명으로 입금 필요

8) 현장실사 일정 조율 (인증원)

- 각 지원 심사팀장 담당 심사원이 업체에 연락하여 심사 일정 조율

- 인증, 연장심사인 경우에 일정 조율하여 심사일시는 불시로 진행

- 심사는 반드시 정해진 '민원처리기간' 내 최종 결과를 알림(인증 심사 60일, 토·일요일과 공휴일 제외)

- 심사 일정 조율 중 업체의 부득이한 사유 발생 시(화재, 공사 미완료, 설비 입고 지연 등) 업체는 인증원에 '민원처리중단' 요청 가능

9) 현장실사 진행 (인증원, 업체)

- 심사원 2명 심사 진행

- 인터뷰를 통해 업체 현황을 파악하며(생산유형, 주요공정, 매출액 등) 원활한 심사와 HACCP 기준 이행 여부를 파악하기 위해 많은 질문이 있을 수 있음

- 심사 총평 후 결과를 당일 알려주며 몇 가지 사항에 대한 서명 후 종료

2

영업장 인·허가 사항 확인

해썹(HACCP) 인증을 받기 위해서는 인증 받고자 하는 업종의 영업 허가가 완료되어야 하며 허가 된 면적 내에서 작업(제조·가공 등)이 이루어져야 합니다.

1) 현장확인 및 구비서류 준비

- 대지 용도 확인

- 업종별 시설기준 준수

- 신축의 경우 설계부터 HACCP 기준 적용

- 구비서류 누락없이 준비

2) 해당 관청에 영업 인·허가 신청

- 관할 관청(지자체)에 영업 허가 신청

- HACCP 인증 신청 확인증 제출이 필요할 수도 있음(한국식품안전관리인증원 해당 지원에 발급 신청)

3) 영업 인·허가 완료 상태 확인

- 식품안전나라에서 영업 인·허가 완료 여부 확인 가능

4) 품목제조보고 신청

- 제조하고자 하는 식품의 품목제조보고 신청

1. 영업 허가를 위한 현장 및 구비서류 준비

<현장>

➀ 영업하고자 하는 대지의 용도가 가공장 건축에 적합한지 확인

* 농지 또는 개발 예정지의 경우 공장 신축이 불가할 수 있으니 사전에 확인 철저

➁ 법 시행규칙 ‘업종별 시설기준’에 적합하도록 설계 및 건축

➂ 신축의 경우 설계 시부터 HACCP 적용 기준에 맞는 이동동선 설정과 적절한 분리·구획·구분이 중요

<서류>

➀ 영업허가신청서

➁ 사업계획서(시설내역, 배치도 등)

➂ 건축물대장 및 임대차계약서

➃ 위생교육수료증

➄ 보건증 등

2. 관할 지자체에 영업 인허가 신청(시도 또는 시군구에 영업 허가 신청)

- HACCP 의무적용 업종의 경우 영업 허가 신청 시 해당 관청에서 ‘HACCP 인증 신청 확인증[붙임1]’ 제출을 요구할 수 있음(‘HACCP 인증 신청 확인증’은 한국식품안전관리인증원 해당 지원에 신청하여 발급)

* HACCP 인증 신청 시 필요한 제출서류 목록과 동일(단, 영업허가증은 제외_아직 허가 전 이므로)

* 문자 발송은 서류접수 완료와 동시에 자동으로 발송되는 시스템으로 인증 신청 서류에 수수료 안내 받을 휴대전화 번호 명확히 기재 필요(번호 오타 주의)

** 문자 발신 번호는 각 지원의 담당자 내선 번호로 표시되며 스팸문자 등으로 오인하여 지나치지 않도록 주의(수수료 안내 문자를 삭제하여 정확한 수수료 금액이 생각나지 않는 경우 해당 지원에 연락하여 재안내 받을수 있음)

5-2. 심사 수수료 입금(업체→인증원 / 계좌이체만 가능)

해당 지원 계좌로 수수료 입금하며 계좌이체만 가능합니다(카드 결제 불가). 입금 확인 후 ‘계산서’를 발급해 드립니다(계산서 우선 발행 불가). 입금자는 반드시 ‘업체명’으로 입금해 주시기 바랍니다.

* 수수료 입금 관련 주의사항

➀ 한국식품안전관리인증원은 부가가치세 면세 사업장으로 ‘세금계산서’가 아닌 ‘계산서’로 발행

➁ 문자 발송된 수수료 금액 정확하게 확인 후 입금[오입금 사례: 1) 수수료 810,000원(81만원)을 81,000(8만 1천원)으로 입금한 사례 2) 식육가공업과 식육포장처리업 동시 신청 시 수수료 안내 문자가 각각 발송되어(식육가공업 1번, 식육포장처리업 1번) 총 2번의 수수료 입금이 필요하나 1번만 입금된 사례]

➂ 수수료 안내 문자를 삭제한 경우 재안내 문의는 해당 지원에 연락하여 요청

➃ 수수료 입금은 안내 문자 수신 후 입금(서류 신청과 동시에 입금 및 서류 신청 전 수수료 먼저 입금 지양)

6. 현장실사 일정 조율

- 각 지원 심사일정 담당 심사관이 업체에 연락하여 심사 일정을 조율 합니다

* 인증·연장심사만 사전에 일정 조율하며 조사평가는 불시평가로 진행

- 현장실사 일정은 심사 신청 접수 순서대로 진행됨을 원칙으로 합니다

- 심사는 반드시 정해진 ‘민원처리기한’ 내에 최종 결과가 확정 되어야 합니다(인증심사 40일, 연장심사 60일 / 단 토·일요일과 공휴일은 민원처리기한 산정에서 제외)

- 심사 일정을 민원처리기한 종료일에 임박하게 희망하면 심사 결과가 ‘보완’일 경우 보완기간이 촉박할 수 있음(유효기간 만료일이 있는 연장심사에 특히 주의)

- 심사일정 조율 중 업체의 부득이한 사유 발생 시(화재, 공사 미완료, 설비 입고 지연 등) 업체는 인증원에 ‘민원처리기한중단’ 요청 가능합니다

7. 현장실사 진행

- HACCP 인증심사는 인증원 심사관 2명이 심사 진행 합니다

- 인터뷰를 통해 업체 현황을 파악하며(생산유형, 주요 공정, 매출액 등) 원활한 심사와 HACCP 기준 이행 여부를 파악하기 위해 많은 질문이 있을 수 있습니다

* HACCP 평가표에도 팀 구성원이 HACCP의 개념과 원칙, 역할에 대해 충분히 이해하고 있는지 평가하는 항목이 있으며 CCP 모니터링 방법, 한계기준, 개선조치 방법 등에 대해 질문이 이루어짐

- 심사관은 노트북을 사용하여 심사 진행합니다(업체 운영 현황 및 심사 지적사항을 입력하고 당일 심사 결과를 알려줍니다. 또한 노트북을 이용해 당일 심사 관련 전자서명을 합니다)

- 총평(심사 결과, 지적사항, HACCP 인증 시 주의사항, 다음년도 심사에 대한 안내 등) 후 심사 종료 됩니다

2. HACCP 인증심사 업무처리 절차

심사 수수료 납부 확인이 되면 심사 신청 접수가 완료되며 인증원 해당 지원에서 심사 일정 수립 후 현장실사를 진행하게 됩니다.

※ 신청 접수 완료 후 현장실사까지 소요되는 기간은 각 지원 별로 다르며(지원 심사관 수, 심사 물량 등 차이가 있음) 정해진 민원처리기한 내 최종 결과 알림(일반적으로 신청 접수 후 현장실사까지 3~4주 정도 소요)

HACCP 인증심사 업무처리 절차

1. 인증(연장)심사 신청 -> 신청 서류 검토 -> 서류 미비 -> 보완(15일 이내) -> 미보완 시 종결

2. 인증(연장)심사 신청 -> 신청 서류 검토 -> 서류 미비 -> 보완(15일 이내) -> 보완 시 현장실사 -> 평가, 판정 -> 이후 적합, 보완, 부적합 단계에 따라 절차가 진행됨.

3. 인증(연장)심사 신청 -> 신청 서류 검토 -> 적합 -> 현장실사 -> 평가, 판정 -> 이후 적합, 보완, 부적합 단계에 따라 절차가 진행됨.

4-1. 평가, 판정 이후 적합인 경우 -> 인증서 작성 -> 인증서 교부

4-2. 평가, 판정 이후 보완인 경우 -> 3개월 이내 보완 시 인증서 작성 -> 인증서 교부

4-3. 평가, 판정 이후 보완인 경우 -> 미보완 시 종결

4-4. 인평가, 판정 이후 부적합인 경우 -> 종결
